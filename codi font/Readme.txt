Si voleu executar el codi, si us plau llegiu atentament i cuidadosament aquest fitxer...

Per executar el programa procoreu seguir els seguents pasos:
1. Modificar el fitxer "globalVariables.py" per tal de definir el flux d'execucio que volgueu. El fitxer es prou clar per entendre que fa cada variable donat que estan definides les opcions posibles que pot pendre qualsevol delles.
2. Executar main.py

PD: Si no sespecifiquen be les variables globals el programa llenca una excepcio, cal llegirla per entendre quin ha sigut el motiu. Per altre banda, si no especifiqueu be les rutes dels datasets la maquina no sabra on trobar el fitxer que desitjeu.

Fitxers del directori:
data.py: codi font sobre el tractament de les dades dun dataset
elbowtest.py: codi font del metode del colze
globalVariables: variables globals sobre lexecucio del main.py
main.py: codi font que executa el programa inicial
kmeans.py: codi font dels 2 kmeans
randindex.py: codi font de la mesura de qualitat externa Rand Index
validation.py: codi font de la validacio de dades amb etiqueta
